
## Build Android Package
```bash
$ androidjs b
```
APK will be in `./dist/`

## Start Local DEV Server
```bash
$ npm run start:dev
```
Open `./views/` in browser.

## Cloud Instance
Open https://energychain.github.io/corrently-current/views/ in browser.
